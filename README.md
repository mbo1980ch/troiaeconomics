# Simple Landing Page

Files included:
- `index.html`
- `styles.css`

Expected assets:
- `logo.png` → place your logo file here
- `background.jpg` → place your full-screen background image here

## How to use
1. Add your `logo.png`
2. Add your `background.jpg`
3. Open `index.html` in a browser

## GitHub deployment
1. Create a new GitHub repository
2. Upload these files to the repository root
3. Enable GitHub Pages from the repository settings if you want to publish it
